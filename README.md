# Student Records Query Program (Python)

A menu-driven console application that reads a tab-delimited student records file and allows users to query and summarize the data.

## Overview
This program demonstrates core Python programming concepts, including file parsing, nested data structures, user input validation, and formatted console output.

## Features
- Display all student records
- Search students by last name prefix
- Search students by graduation year
- Generate a summary report by graduation year (counts and percentages by degree program)

## Files
- GroupJ-FinalCode.py — main program script
- students.txt — sample input data file (tab-delimited)

## How to Run
python GroupJ-FinalCode.py

When prompted:
- Enter a filename, or
- Press Enter to load the default students.txt

## Input File Format
The input file must be tab-delimited and include a header row.

Example header:
ID	Last	First	GradYear	GradTerm	DegreeProgram

## Example Menu
1 - Display ALL students
2 - Search by last name prefix
3 - Search by graduation year
4 - Program Summary by graduation year
quit - Exit program

## Skills Demonstrated
- File input/output and validation
- Dictionary-based data modeling
- Modular function design
- Console-based reporting
